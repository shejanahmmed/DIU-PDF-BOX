About DIU PDF BOX

👨‍💻 Meet the Creator

Hi! I'm Farjan Ahmmed, a Software Engineering student at Daffodil International University pursuing my BSc degree.

As a fellow DIU student, I understand the daily struggles of creating properly formatted academic documents and managing academic documents.

🎯 Why DIU PDF BOX?

Every semester, DIU students spend countless hours formatting assignment covers, merging PDFs and organizing documents. I created DIU PDF BOX to solve these common problems:

  • Time-consuming formatting - No more manual cover page creation
  • Inconsistent layouts - Professional, standardized templates
  • File management hassles - Easy PDF merging and organization
  • Mobile accessibility - Works perfectly on phones and tablets

✨ Key Features

Assignment Covers

Generate professional DIU assignment covers instantly

PDF Merging

Combine multiple PDFs into one document

Mobile Friendly

Works seamlessly on all devices

No Registration

Use all features without creating an account

🚀 Technology Stack

DIU PDF BOX is built using modern web technologies:

  • Frontend: HTML5, CSS3, JavaScript
  • PDF Processing: PDF-lib library
  • UI/UX: Responsive design with Font Awesome icons
  • Hosting: GitHub Pages for fast, reliable access
